## Getting started (local development)

1. Clone the repo

   git clone https://github.com/milkosamuel/portfolio-web.git
   cd portfolio-web

2. Install dependencies

   npm install

3. Run the development server

   npm run dev

4. Build for production

   npm run build
   npm start
